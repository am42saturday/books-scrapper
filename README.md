# Book Scraper

Проект для автоматического сбора данных о книгах с сайта [Books to Scrape](http://books.toscrape.com/).

## Функциональность

- **Парсинг данных о книгах** с отдельных страниц (`get_book_data`)
- **Сбор всех книг** со всех страниц каталога (`scrape_books`)
- **Автоматическое обновление данных** по расписанию с помощью `schedule`
- **Автотесты** с использованием `pytest`

## Запуск
### Запуск парсера вручную
```bash
python scraper.py
```

### Запуск по расписанию (ежедневно в 19:00)
```bash
python scheduler.py
```

### Запуск тестов
```bash
pytest -v
```
